## Description

**Lobe** est un site web en cours de développement.  
L’objectif est de proposer une interface moderne et intuitive, avec de nouvelles fonctionnalités à venir et une expérience responsive sur tous les écrans.

Fonctionnalités à venir :  

- Ajout de nouvelles fonctionnalités interactives  
- Support **responsive** pour mobile, tablette et desktop  
- Amélioration continue de l’expérience utilisateur  

---

## Stack utilisée

- **HTML5** – structure du site  
- **CSS3** – mise en page, animations et design responsive  
- **JavaScript** – interactivité et navigation dynamique  

---

## Fonctionnalités actuelles

- Barre de navigation fixe et interactive  
- Navigation entre les sections du site  
- Sections pré-remplies : Overview, Examples, Tour, Blog et Help
